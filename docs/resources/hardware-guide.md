# Hardware Guide

Guide to hardware components for Physical AI projects. All projects can be completed in simulation.

## Recommended Components

| Component | Price Range | Use Case |
|-----------|-------------|----------|
| Raspberry Pi 4 | $45-55 | Main computer |
| Arduino Uno | $20-25 | Motor control |
| Servo motors | $3-15 | Robot joints |
| Ultrasonic sensor | $2-5 | Distance sensing |
| Camera module | $25-30 | Computer vision |

## Budget Options

- **Simulation Only**: $0 - Complete all projects in software
- **Starter Kit**: $50-100 - Basic sensors and Arduino
- **Full Kit**: $200-300 - Complete robot platform
